# 🎬 Film Finder Bot

A Telegram bot that finds movies by description using AI

## Features
- Finds movies by description using GPT
- Movie posters and trailers via TMDB API
- Similar movie recommendations
- Movie of the day and evening picks

## Tech Stack
Python, aiogram, OpenAI API, TMDB API

## How it works
User describes a movie → GPT identifies the title → TMDB returns poster, rating and trailer
